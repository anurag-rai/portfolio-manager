Listing all API Endpoints:
-------------------------------------------------------

GET /portfolio
Return all entire portfolio with trades

-------------------------------------------------------

GET /portfolio/:id
Returns the portfolio with the id

-------------------------------------------------------

GET /portfolio/:portfolio/holdings
Get holdings in an aggregate view

-------------------------------------------------------

GET /portfolio/:portfolio/returns
Get cumulative returns

-------------------------------------------------------

POST /api/trades/:portfolio/
{
	stock: "stockName",
	quantity: integer,
	action: "buy/sell"
}
{
	success: boolean,
	data: Trade object - has id,stock,quantity, action, rate, timestamp
}

-------------------------------------------------------
